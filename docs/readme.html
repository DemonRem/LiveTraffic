Essential Installation

The simplest way to get it up and running is as follows. If you have XSquawkBox already and want to run both plugins in parallel please refer to the comprehensive installation instructions .

LiveTraffic plugin

The LiveTraffic plugin folder is to be copied into X-Plane's Resources/plugins folder, the same as with other plugins.

XSquawkBox

Download from here, unzip, and move the entire XSquawkBox folder under X-Plane's Resources/plugins folder, i.e. parallel to LiveTraffic. No further configuration is necessary for the purposes of LiveTraffic.

The LiveTraffic installation archive contains one file LiveTraffic/Resources/related.txt, which is a minimal enhanced version of the one which comes with the XSquawkBox package and is supposed to be a drop-in replacement. It matches the A320 Neo with the A320 model and the Empbraer 175 with the 170 models, which otherwise would be displayed with an A320 model as that is the default in case of no match. Copy it over the existing version in XSquawkBox/Resources.
